# Hotel Management Desktop Application

A desktop application for managing hotel operations.  
This project was developed as an academic assignment to practice **Fullstack Development** with **Java** and **Java Swing**.

## 📌 Project Information
- **Team size:** 4  
- **Role:** Fullstack Developer  
- **Duration:** Dec 2025 – Mar 2026  
- **Languages/Frameworks:** Java, Java Swing  

---

## 📌 Overview
The system is designed to manage hotel operations including room bookings, customer data, and administrative tasks.  
It follows a UML-driven design and implements a **Three-layer Architecture**.

---

## 📌 Features
- <mark>Authentication</mark>  
- <mark>Admin Dashboard</mark> for managing rooms and customers  
- <mark>Booking Management</mark>  
- <mark>Statistics</mark> and reporting  

---

## 📌 System Design
- Designed <mark>UML Class Diagram</mark> to model system entities and dependencies  
- Implemented <mark>Three-layer Architecture</mark>  
- Database schema in <mark>Microsoft SQL Server</mark>  

---
